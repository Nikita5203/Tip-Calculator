# Tip-Calculator

A simple web application that calculates the tip amount and total bill based on the bill amount and selected tip percentage.

## ✨ Features

- Input bill amount
- Select tip percentage
- Calculate tip amount
- Calculate total bill
- Instant result display
- Clean and responsive UI

## 🛠️ Technologies Used

- HTML5
- CSS3
- JavaScript
